# CISP360-Final-Mansion Escape Game
## Overview
Console based C++ adventure game.
Player moves through rooms inside a mansion and tries to reach the exit.

## Game Goal

Start at the entrance.
Move through connected rooms.
Reach the exit in the bedroom area to win.

## How It Works

* Game loads a fixed map of 5 rooms
* Each room has a name, description, and exits
* Player inputs movement commands
* Game updates location based on valid moves
* Some rooms trigger monster encounters
* Player must survive and continue moving
* Exit condition ends the game

## Controls

* N move north
* S move south
* E move east
* W move west
* L look around room details

## Combat System

* Monsters appear in specific rooms
* Player chooses attack option
* Simple success based interaction
